# Transactions Dashboard By David Abu

A serverless application for real-time monitoring of transactions conducted by agent clients.

## Client Side Technologies

- **React**
- **React Router**
- **Redux**
- **Events**
- **Material UI**
- **Styled Components**
- **Axios**

## Backend Technologies

- **Firebase**
  - Firestore Database
  - Storage
  - Authentication
